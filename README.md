# Advent of Code 2024

Solutions for Advent of Code 2024 challenges in Python.

## Structure

- `day1.py`: Solution for Day 1 - Historian Hysteria
- `day2.py`: Solution for Day 2 - Red-Nosed Reports
- `day3.py`: Solution for Day 3 - Mull It Over

## Running Solutions

Each solution can be run independently:

```bash
python dayN.py  # where N is the day number
```

Make sure to place your input in a file named `inputN.txt` in the same directory as the solution.
